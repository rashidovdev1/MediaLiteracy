# Mediasavodxonlik elektron darsligi

Ushbu loyiha foydalanuvchilarga mediasavodxonlikni oshirish uchun videodarslar, adabiyotlar va testlar taqdim etuvchi elektron platforma hisoblanadi.

## Loyihaning maqsadi

Platforma orqali foydalanuvchilar mediasavodxonlik bo‘yicha bilimlarini mustahkamlashlari, videodarslarni tomosha qilishlari, adabiyotlar bilan tanishishlari va testlar yordamida o‘z bilimlarini sinovdan o‘tkazishlari mumkin.

## Texnologiyalar

- Python 3.x
- Django 4.x
- HTML, CSS, JavaScript
- SQLite (yoki boshqa ma’lumotlar bazasi)
